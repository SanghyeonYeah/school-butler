# student-butler

통학하는 학생의 하루를 관리해주는  
**집사형 캐릭터 오버레이 어시스턴트 모바일 애플리케이션**

---

## 📌 프로젝트 소개

본 프로젝트는 **열품타의 집중 관리 개념**과 **헬로펫 스타일 캐릭터 상호작용**을 결합한 모바일 애플리케이션이다.  
기존 알람·리마인더 앱이 화면을 갑작스럽게 점유하여 학습 흐름과 일상을 방해한다는 문제의식에서 출발했다.

이 앱은 알람과 학교 생활 안내를  
**집사 캐릭터가 말풍선 또는 작은 표시로 조용히 전달하는 오버레이 방식**으로 제공한다.

사용자는 앱 실행 시 로비 화면에서 집사를 설정하고,  
앱 종료 이후에도 화면 위를 돌아다니는 집사를 통해 하루 루틴을 자연스럽게 관리받는다.

---

## ✨ 핵심 특징

- 전체 화면을 가리지 않는 **비침투적 UX**
- 앱 종료 후에도 유지되는 **플로팅 집사 오버레이**
- 알람이 아닌 **보고·안내 중심 인터랙션**
- 통학 학생의 하루 루틴에 특화된 설계

---

## 🧩 주요 기능

### 1. 집사형 오버레이 어시스턴트

- 로비 화면에서 집사 캐릭터 대기 (관리자 역할)
- 앱 종료 후에도 유지되는 오버레이 집사
- 집사 터치 시 최소 메뉴 표시
- 말풍선 기반 알림 (명령 / 보고 / 안내 톤)
- 화면을 가리지 않는 설계

#### 집사 반응 시스템
- 알림 확인 여부에 따른 말투 변화
- 일정 미이행 시 재안내 또는 걱정 표현
- 신뢰도 / 호흡도 개념 기반 상호작용
- 집사 디자인·말투는 추후 확장 예정

---

### 2. 알람 · 일정 관리

통학 학생의 하루 루틴을 집사가 관리

- 기상 알람  
  > “기상 시간입니다. 1교시까지 45분 남았습니다.”
- 등교 준비 알림
- 학교 종소리 알림
- 수업 시작·종료 리마인더
- 시험 일정 및 준비 알림
- 과제·수행평가 개인 리마인더

※ 모든 알림은 **집사가 조용히 안내**  
(전체 화면 점유 없음)

---

### 3. 학교 정보 요약 제공

집사가 필요 시 핵심 정보만 요약 안내

- 오늘 급식 정보
- 오늘 시간표 요약
- 학생 공지사항 핵심 요약
- 시험 고사실 정보
- 시험 시간표

예시:
> “오늘 수학 수행평가가 있습니다. 4교시입니다.”

---

### 4. 메신저 · 소셜 (선택 기능)

- 간단한 메신저
- 친구 추가
- 친구 집사 상태 확인
- 상태 공유 (집중 중 / 졸림 / 시험 기간 등)

※ 핵심은 **관리 기능**, 소셜 기능은 최소화

---

## 사용자 흐름

1. 앱 실행
2. 집사 캐릭터 설정
3. 기상·학교·일정 알람 설정
4. 앱 종료
5. 오버레이 집사가 일상 속에서 관리
6. 필요 시 집사 터치 → 정보 확인
7. 친구 기능 선택적 사용

## 기술 스택

- **Framework**: Flutter
- **Backend**: Python
- **Platform**: Android / iOS
- **Data**
  - 리로스쿨 API (가능한 경우)
  - 자체 일정·알람 서버

### 플랫폼별 특징
- **Android**: 시스템 오버레이 적극 활용
- **iOS**: 위젯 및 알림 중심 설계 (플랫폼 제한 고려)

## UI / UX 방향성

- “관리받는 느낌”의 안정적인 UX
- 열품타 사용자에게 익숙한 집중 중심 설계
- 알람이 아닌 보고·안내 중심 인터랙션
- 캐주얼하지만 과하지 않은 캐릭터 디자인
- 학생의 하루 루틴을 대신 기억해주는 구조
